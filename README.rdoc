= rsdl

rsdl - SDL initialized ruby interpreter

== Requirements
- ruby 1.8 or later
- libsdl

== Installation

  ruby extconf.rb
  make
  sudo make install

== Usage

  Run your script with rsdl instead of ruby.

  rsdl myscript.rb
